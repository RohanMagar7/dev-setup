## Overview

This project aims to simplify the process of setting up a development environment on Ubuntu by automating the installation and configuration of commonly used development tools and software.

## Prerequisites

- Ubuntu 20.04 LTS or later
- `sudo` privileges
- Active internet connection
- At least 10GB of free disk space

## Quick Start

```bash
# Clone the repository
git clone https://github.com/RohanMagar7/Full-Dev-Setup-Ubuntu-.git

# Navigate to the directory
cd Full-Dev-Setup-Ubuntu-

# Run the setup script (coming soon)
# ./setup.sh
```

## What's Included

This setup includes configurations and installations for:

- **Version Control**: Git, GitHub CLI
- **Programming Languages**: Python, Node.js, Java, Go, Rust
- **Text Editors & IDEs**: VS Code, Vim
- **Database Systems**: PostgreSQL, MySQL, MongoDB, Redis
- **Containerization**: Docker, Docker Compose
- **Development Tools**: curl, wget, build-essential, and more
- **Shell Enhancements**: Zsh, Oh My Zsh, custom themes

## Troubleshooting

### Common Issues

- **Permission Denied**: Ensure you have `sudo` privileges
- **Package Not Found**: Update your package lists with `sudo apt update`
- **Network Issues**: Check your internet connection

For more help, please open an issue on GitHub.
